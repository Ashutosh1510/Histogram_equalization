# Histogram_equalization

## Required Packages
1. Python3
2. Opencv 
3. Numpy

# Instructions to run the program
--git clone or download the zip folder

- cd to the Source folder 

1. Histogram Equalization:
```
python3 hist_eq.py
```
2. Adaptive Histogram Equalization:
```
python3 ahe.py
```
